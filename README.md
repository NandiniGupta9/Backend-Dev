# Backend-Dev
# 🚀 Backend Development Journey (Node.js + Express + MongoDB)

This repository documents my structured learning journey in backend development based on a comprehensive syllabus covering Node.js, Express, MongoDB, authentication, and deployment.

---

## 📚 Course Modules

### 🧩 Module 1: Backend Fundamentals

* **Unix Commands & Git/GitHub**

  * Version control, branching, merging, conflict resolution

* **Node.js Basics**

  * Core modules (`http`, `fs`, `path`, `os`)
  * Event loop & server creation

* **File System Operations**

  * Reading/writing files (sync & async)
  * Directory handling & error handling

* **Express.js Fundamentals**

  * Server setup, routing, middleware
  * Templating (EJS), static files

* **RESTful APIs**

  * GET, POST, PUT, DELETE
  * CRUD operations & API design

---

### 🧩 Module 2: Advanced Backend Concepts

* **Databases (SQL vs NoSQL)**

  * MongoDB basics, Mongoose ODM
  * Schema design & CRUD operations

* **Middleware, Sessions & Cookies**

  * Custom middleware
  * Client-side vs server-side storage

* **Authentication & Authorization**

  * Password hashing using bcrypt
  * Session-based authentication

* **Security**

  * MongoDB Injection, XSS protection
  * Input validation & sanitization

* **Deployment**

  * Deploying apps (Vercel/Heroku)
  * MongoDB Atlas integration
